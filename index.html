umx

umx is an R package relentlessly focussed on making OpenMx doing Structural Equation Modeling in OpenMx quicker, easier, and more powerful. It's easy to learn and easy to get print-ready results from models. Sweet huh?

Install it now and run a model!

You can install the package in R as follows:

install.packages("devtools")
library("devtools")
install_github("umx", username = "tbates")
library("umx")
?umx
# on windows you might need
# install_github("umx", username = "tbates", args = "--no-multiarch")
# On the newest version of devtools, this will work
# install_github("tbates/umx")
